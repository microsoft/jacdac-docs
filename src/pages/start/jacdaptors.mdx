## Jacdaptors

Jacdac can provide a convenient way to add functionality to existing electronics prototyping platforms including micro:bit, Raspberry Pi and Arduino, via _Jacdaptors_. As shown above, the current set of Jacdaptors includes:
- [KittenBot's Jacdaptor](/devices/kittenbot/jacdaptorformicrobitv2v10) for the [micro:bit V2](/devices/microbit-educational-foundation/microbitv2/), programmable via [MakeCode](/clients/makecode)
- [Raspberry Pi JacHat](/devices/microsoft-research/jmspibridgev37), programmable via [JavaScript](/clients/javascript)
- [Jacdac to USB bridge](/devices/microsoft-research/jmbrainf441v02) to a laptop/desktop programmable via [JavaScript](/clients/javascript)

In the above cases, the Jacaptor bridges the Jacdac bus to an existing programmable device (micro:bit, Raspberry Pi, laptop/desktop), or brain.
