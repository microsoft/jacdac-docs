# Blinky

A blinking blue light

```python filename=blinky.py copy
from jacdac import Bus
from jacdac.led import LedClient
from time import sleep

bus = Bus()
led = LedClient(bus, "led")
led.brightness = 0.5
# fade between colors
while True:
    # blue
    led.set_all((0,0,255))
    sleep(1)
    # off
    led.set_all((0,0,0))
    sleep(1)
```
