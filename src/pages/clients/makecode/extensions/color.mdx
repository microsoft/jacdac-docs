# Color detection

The color detection extension provides three methods
(`red`, `green`, and `blue`), each of which returns a value
in the range of 0.0 to 100.0

```blocks
basic.forever(() => {
    let red = modules.color1.red()
    let blue = modules.color1.blue()
    let green = modules.color1.green()
    if (green > 50 && red + blue < 20) {
        led.stopAnimation()
        basic.showString("G")
    } else {
        basic.clearScreen()
    }
})
```
