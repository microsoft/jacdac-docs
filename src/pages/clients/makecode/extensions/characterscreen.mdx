# Character Screen

The characterscreen extension controls a simple text display, such as a LCD screen or Braille dot matrix.

The following program is a counter where button A increments by 1 and button B decrements by 1.

```blocks
input.onButtonPressed(Button.A, function () {
    count += 1
    modules.characterScreen1.setLine(0, count)
})
input.onButtonPressed(Button.B, function () {
    count += 0 - 1
    modules.characterScreen1.setLine(0, count)
})
let count = 0
count = 0
modules.characterScreen1.setLine(0, count)
```
