# Air Pressure

The air-pressure extension (barometer) reports the air pressure in hPa
(the abbreviated name for a hectopascal pressure unit,
which is exactly equal to a millibar).
The `pressure` method returns the current barometric pressure.

The following program outputs the pressure whenever the pressure changes by 0.2 hPa:

```blocks
modules.airPressure1.onPressureChangedBy(0.2, function() {
    led.stopAnimation()
    basic.showNumber(modules.airPressure1.pressure())
})
```
