# Gamepad

The gamepad extension supports a two axis directional joystick
via methods `x` and `y` that return values in the range -100.0 to +100.0.
Events are generated by various motions of the joystick, as well
as optional buttons that may be associated with the joystick.

## Coordinates

```blocks
basic.forever(() => {
    plot(modules.gamepad1.x(), modules.gamepad1.y())
})

function plot(x: number, y: number) {
        basic.clearScreen();
        let dispX = 2;
        let dispY = 2;
        if (x < -30)
            dispX--;
        if (x < -60)
            dispX--;
        if (x > 30)
            dispX++;
        if (x > 60)
            dispX++;
        if (y < -30)
            dispY--;
        if (y < -60)
            dispY--;
        if (y > 30)
            dispY++;
        if (y > 60)
            dispY++;
        led.plot(dispX, dispY);
}

```

## Events

```blocks
modules.gamepad1.onEvent(jacdac.GamepadEvent.ButtonsChanged, function () {
    if (modules.gamepad1.isButtonDown(jacdac.GamepadButtons.Left)) {
        led.stopAnimation()
        basic.showArrow(ArrowNames.West)
    } else if (modules.gamepad1.isButtonDown(jacdac.GamepadButtons.Right)) {
        led.stopAnimation()
        basic.showArrow(ArrowNames.East)
    } else if (modules.gamepad1.isButtonDown(jacdac.GamepadButtons.Up)) {
        led.stopAnimation()
        basic.showArrow(ArrowNames.North)
    } else if (modules.gamepad1.isButtonDown(jacdac.GamepadButtons.Down)) {
        led.stopAnimation()
        basic.showArrow(ArrowNames.South)
    }
})
```
