# HID Joystick

The HID joystick extension can be used to control a HID mouse, via a Jacdac module
that implements HID (generally communicating to a desktop/laptop over USB). Use the
[JM Brain F4 v0.2](/devices/microsoft-research/jmbrainf441v02) or [JM Brain RP2040 v0.1](/devices/microsoft-research/jmbrainrp204059v01) module,
loaded with the HID Joystick firmware that is available via the same links.

## Joystick buttons

The state of a buttons can be modified using `setButton`:

```blocks
input.onButtonPressed(Button.A, function () {
    led.stopAnimation()
    modules.hidJoystick1.setButton(0, true)
    basic.showIcon(IconNames.Heart)
    modules.hidJoystick1.setButton(0, false)
    basic.clearScreen()
})
```

## Joystick analog inputs

The state of a axis can be modified using the `setAxis`:

```blocks
basic.forever(function () {
    modules.hidJoystick1.setAxis(0, input.acceleration(Dimension.X) / 10)
})
```
