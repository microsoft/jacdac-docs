# Temperature

The temperature extension reports the temperature in degrees Celsius.
The `temperature` method returns the current temperature.

The following program outputs the current temperature whenever it changes by
one degree Celsius:

```blocks
modules.temperature1.onTemperatureChangedBy(1, function () {
    led.stopAnimation()
    basic.showNumber(modules.temperature1.temperature())
})
```
